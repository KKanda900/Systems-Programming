Description:

With this test case we can see how accurate the test is when all the files in the directory are unrelated. test0.txt is just "a b c d e", test1.txt is a short story and test2.txt is random gibberish.

We see that the random gibberish test2 does have JSD of 0.30 with the other two texts, which is the max value for completely different files (1*log_10(1/0.5)=0.301).
JSD between test0 & test1 is slightly lower because of the token "a".
